DECIMAL TO BINARY

Calculator

method:

  1. Divide the number by 2.
  2. Get the integer quotient for the next iteration.
  3. Get the remainder and multiply by 2 for the binary digit.
  4. Repeat the steps until the quotient is equal to 0.

example:

Convert the Decimal number 512 to the Binary number system

  1. 512 / 2 = 256.5, the remainder is 0 => 0️⃣ = 0
  2. 256 / 2 = 128, we multiply the remainder by 2 which is 0.5 *2 => 1️⃣ = 1
  3. 128 / 2 = 64, the remainder is 0 => 2️⃣ = 0
  4. 64 / 2 = 32, the remainder is 0 => 3️⃣ = 0
  5. 32 / 2 = 16, the remainder is 0 => 4️⃣ = 0
  6. 16 / 2 = 8, the remainder is 0 => 5️⃣ = 0
  7. 8 / 2 = 4, the remainder is 0 => 6️⃣ = 0
  8. 4 / 2 = 2, the remainder is 0 => 7️⃣ = 0
  9. 2 / 2 = 1, the remainder is 0 => 8️⃣ = 0
  10. 1 / 2 = 0.5, the remainder is 0 => 9️⃣ = 1

the result is 1000000010 = 512
